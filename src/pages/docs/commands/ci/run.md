# glab ci run

Create or run a new CI pipeline

```bash
glab ci run [flags]
```

## Examples

```bash
glab ci run
glab ci run -b trunk

```

## Options

```bash
  -b, --branch string       Create pipeline on branch/ref <string>
      --variables strings   Pass variables to pipeline
```
