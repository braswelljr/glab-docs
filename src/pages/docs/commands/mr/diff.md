# glab mr diff

View changes in a merge request

```bash
glab mr diff [<id> | <branch>] [flags]
```

## Examples

```bash
glab mr diff 123
glab mr diff branch
glab mr diff  # get from current branch
glab mr diff 123 --color=never

```

## Options

```bash
      --color string   Use color in diff output: {always|never|auto} (default "auto")
```
