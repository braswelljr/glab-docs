# glab mr subscribe

Subscribe to merge requests

```bash
glab mr subscribe [<id> | <branch>] [flags]
```

## Examples

```bash
glab mr subscribe 123
glab mr sub 123
glab mr subscribe branch
glab mr subscribe 123 branch  # subscribe to multiple MRs

```
